# Personal Portfolio Android App

### Overview
This is the official mobile application for **Madhav Kumar's** personal portfolio, designed to showcase skills, projects, certifications, internships, and contact information. The app provides a smooth and responsive experience to highlight professional expertise and achievements, helping users navigate easily through sections like "About Me," "Skills," "Projects," and more.

### Features
- **About Me:** Personal introduction and background information.
- **Skills:** A list of technical and soft skills.
- **Certifications & Internships:** Display of earned certifications and completed internships.
- **Projects:** Showcase of personal and professional projects, including descriptions and links to live demos or repositories.
- **Contact:** Integrated form for users to get in touch.
- **Offline Access:** Most sections can be accessed without an internet connection.

### Tech Stack
- **Language:** Java/Kotlin
- **UI Design:** XML, Material Design components
- **Architecture:** MVVM (Model-View-ViewModel)
- **Backend:** Firebase (optional for storing contact form inquiries)
- **Libraries:**
  - Retrofit for networking
  - Glide for image loading
  - Room for local database
  - Firebase Analytics (optional)
